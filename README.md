# ml-iot-detection
IoT device detection project for machine learning class at utep


3 Main branches for the 3 models trained with the dataset.

1. knn-models for knn - Kha
2. neural-network-models for neural networks - Blaine
3. random-forest-models for decision trees / random forest - Vic

Each branch will have a readme file explaining how to run the training,testing, and other important scripts for the models
