# ml-iot-detection
IoT device detection project for machine learning class at utep



All scripts found in src/models/
For multi-headed NN
1. To train the NN, run the file test_multi_nn.py
2. To test the NN, run the file test_multi_nn.py

For 4 separate NNs
1. To train the NNs, run the file train_nn.py
2. To test the NNs, run the script test_nn.py



Data used to train model found in data/processed/
X_matrix.parquet - Input Features
Y_matrix.parquet - Classifcation Features
